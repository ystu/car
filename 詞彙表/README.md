# 詞彙表

這個資料夾用來集中管理逐字稿整理時需要用到的語音辨識詞彙。

## 檔案說明

| 檔案 | 用途 |
| --- | --- |
| `語音辨識詞彙總表.md` | 所有已確認詞彙的總覽，整理逐字稿時優先查這份。 |
| `人名與職稱.md` | 人名、職稱與常見誤辨。 |
| `保固與流程.md` | 保固、稽核、工單、診斷、打卡等流程詞彙。 |
| `車輛術語.md` | 車輛部件、設備、測試方式與維修相關詞彙。 |
| `系統代碼.md` | 引擎代號、系統代碼、配備代碼與車型相關詞彙。 |
| `待確認語音詞彙.md` | 尚未確認、暫不收進正式詞彙表的詞彙。 |

## 使用方式

1. 整理逐字稿時，先查 `語音辨識詞彙總表.md`。
2. 如果要針對某一類詞彙補充資料，再編輯對應分類檔。
3. 還不確定的詞先放到 `待確認語音詞彙.md`，確認後再移到總表與分類檔。
